Style

Grid

The grid provides the foundation for harmoniously positioning elements.

The grid provides a flexible, mobile-first, device-agnostic layout system. It has features to control alignment, order, and flow.

Structure

Designing to the grid creates an experience that facilitates understanding and brings order to the page.

Exponential grid

The width of the grid and content area grows exponentially and the margins always remain at 5%. The width of each column continuiously grows and gutters remain fixed.

Maximum width

The max width is 1600px, with the margins growing exponentially past that point.

Breakpoints

The structure of our grid adapts to different device widths to make use of space as a user's viewport size increases.

Name Width Gutter
xs ≤ 36rem 1rem
sm > 36rem 1rem
md > 48rem 1.5rem
lg > 62rem 1.5rem
xl > 72rem 2rem
xxl > 100rem 2rem

Vertical rhythm

We use a spacing scale based on powers of two that starts at 0.25rem, or 4px for most devices.

Scale rem px
0 0rem 0px
1 0.25rem 4px
2 0.5rem 8px
3 1rem 16px
4 2rem 32px
5 4rem 64px
6 8rem 128px

With CSS grid, we no longer need a grid system or framework to build flexible, responsive layouts.

We have tailored a system of mixins and utility classes to help align and speed the process of developing based on the Doggo design system.

Width

Code snippets for controlling grid width.

Mixins

Below you will find a collection of mixins created to rapidly build common layouts.

@mixin grid-thirds {
  display: grid;
  grid-template-columns: repeat(3,1fr);
}

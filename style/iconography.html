Style

Iconography

Iconography uses images and symbols to represent an object visually.

They communicate a message and should be distinct and informative. Icons should be used sparingly throughout the product to provide clarity and reduce cognitive load on users.

Size

UI icons come in two sizes: 24px and 16px. While most icons occupy a square artboard, some icons may occupy a rectangular artboard where the width or height of the icons is either 24px or 16px.

Usage rem px
Standard 1.5rem 24px
Glyph 1rem 16px

Space

When creating icons, do not add internal padding with Sketch or Illustrator. If the icon needs padding, it can added with CSS.

When using icons, all touch targets need to be 3rem (48px) or higher. Developers can add padding to a touch target with CSS to meet the requirement if necessary.

Color

Interactive UI icons need to pass the same color contrast ratio as typography at a 4.5:1 ratio. UI icons are always a solid color. The color of the icon should depend on the importance of the icon's action.

Action Color
Primary $blue--3
Primary hover $blue--4
Secondary $black--1
Secondary hover $blue--3

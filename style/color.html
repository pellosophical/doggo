Color brings a design to life.

It's used to express emotion and tone, place emphasis, and create associations. Color should always be used in meaningful and intentional ways to establish patterns and create visual cues.

Palette

We've developed a color system that helps users find people, identify status, see actions, to locate help, and to indicate next steps. The consistent use of color keeps cognitive loads low and makes for a unified and engaging user experience.

Contrast

We are committed to complying with AA standard contrast ratios. To do this, choose primary, secondary and extended colors that support usability by ensuring sufficient color contrast between elements so that users with low vision can see and use the interface.

Components

Button

Express what action will occur when a user clicks it.

Buttons are used primarily on action items. Do not use Buttons as navigational elements. Instead, use Links because it takes the user to a new page and does not express an action.

Labels

Button labels are sentence case and are as short as possible, while clearly explaining what happens when the button is pressed.

Variations

Variations can be used for all button types. For consistency and uniformity choose a single button variation and do not mix them.

Getting Started

Developers

The Doggo Component Library provides front-end developers & engineers a collection of reusable HTML and SCSS partials to build websites and user interfaces.

Install

1. Using NPM

$ npm install doggo

2. GitHub Repo

Go to the Doggo Repo and download the latest version of the Component Library.
